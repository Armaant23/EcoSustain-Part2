# EcoSustain

A Flask app to track eco-friendly actions and CO2 savings.

## Download from GitHub

1. Go to: https://github.com/armaant23-dev/EcoSustain-Part2
2. Click the green **Code** button
3. Click **Download ZIP**
4. Extract the ZIP to any folder

## How to Run
1. Open command prompt in folder:  
Navigate to  project folder using:
cd path\to\EcoSustain-Part2
here is mine as a example: cd C:\Users\armaa\Desktop\EcoSustain_Code

4. Create virtual environment:
python -m venv venv

5. Activate it:
venv\Scripts\activate

6. Install requirements:
pip install -r requirements.txt

7. Run the app:
python app.py

8. Open browser:
http://127.0.0.1:5000

## Features

Register and login,
Log eco actions (such as walking and cycling),
Track daily and weekly CO2 savings,
View all logged actions,
